Timer Manager + Async Queue

Опис проєкту

Даний проєкт реалізує:

- менеджер таймерів
- асинхронну чергу задач
- систему pause/resume
- обробку async задач
- статистику черги
- керування timeout та interval

 Основні можливості

 Timer Manager

Менеджер таймерів підтримує:

- створення timeout
- створення interval
- pause / resume
- clear одного таймера
- clearAll
- список активних таймерів
- відображення часу, що залишився

 Async Queue

Черга задач підтримує:

- додавання async задач
- обмеження concurrency
- pause / resume
- callback пустої черги
- статистику виконання
